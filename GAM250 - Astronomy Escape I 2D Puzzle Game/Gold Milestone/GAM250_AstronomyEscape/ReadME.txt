Level Editor
--------------------------------------------------------------------------------------------------------------------------------
To access the level editor	-ALT - L Key

Creating a new game board
	Check “Create New Board” under the “Select Mode”
	Select the desired size of the board throught the "Stack", "Slice" and "TileSize" under the “Generate Sample Board” filter
	Select “Generate” to proceed on generating the new board.
	A new board will be generated and a message is displayed under the “MESSAGE BOX” filter on the successful creation of the text file. 

Loading an existing game board
	Check “Load Existing Board” under the “Select Mode”
	By pressing the drop down box under the “Load Existing” filter, you are able to select an existing game board
	Select “Load” to load the selected game board and it will be displayed

After Creating new Board or Loading an existing board

	“Object Data”, “Texture Data”, “Direction Data” and “Save Objects” filters will be displayed on the user interface.
	“Object Data” filter, select a list of objects to be added to the board.

	*After selecting the object data*
	
	“Directional Data” filter, select the direction of the object.
	“Texture Data” filter, select texture of the object. (only some object contain multiple textures, such as wall)
	An image preview will be displayed on the user interface depending on what you had selected. 
	"Teleporter ID" filter, select the same teleporter id to link both of the teleporter together.
	"Lever ID" filter, select the same lever door and switches id to link both of them together.

	“Add Object”, to add objects that you had selected by clicking the tiles on the board
	“Delete Object”, to delete objects by clicking the tiles on the board
	“Clear” , clear the objects that had been added onto the board
	“Save” , save the current board. 

	“MESSAGE BOX” will be displaying the information such as the board has been saved successfully, adding objects that are outside of game space, etc.
